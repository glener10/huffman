# Demo

 
1- Enter the file directory.

2- Place the .txt file to be compressed or the .hx file to be uncompressed in it.

3- Run as follows: 

```bash
# To compress
$ ./huffman c inputname.txt output.hx            
```

```bash
# To decompress
$ ./huffman d inputname.hx output.txt
```
